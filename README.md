# First Head Backend

> Build A recipebook App Started with Node, Express and Postgres Using Sequelize 

# Requirement

* postgres
* node (I use v7)
* pgadmin (optional)

# Install & Start

Using `npm`

```shell
cd pathtorepo/
npm install --save
```

Using `nodemon` to watch any change of js file in public/js

```shell
npm run-script mon
```
